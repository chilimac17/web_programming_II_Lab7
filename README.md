To get the server and client running follow the steps below:

cd server
npm i

Once the packages are installed move to the react_client folder and use the same steps

cd ../react_client
npm i

Once both are installed go into the server folder and run the command below to start the server

cd ../server
npm start

After the server starts open another terminal and cd into the react_client folde. run the command below to start the client
cd react_client
npm run dev
